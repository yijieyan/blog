    最近在做项目多次碰到用post去下载文件,开始让后端改,最后总是各种原因,也没改,那就只能自己想办法了。

    # 1.首先更改封装的post方法

      post(url, bodyParams = {}, params = {}, config = {}) {
    if (Object.keys(params).length > 0) {
      for (const key in params) {
        url += `&${key}=${params[key]}`
      }
    }
    if (Object.keys(config).length > 0) {
      return Http.post(url, bodyParams, config).then(res => res.data)
    } else {
      return Http.post(url, bodyParams).then(res => res.data)
    }
  }

    # 2. 修改response拦截器

     if (response.headers['content-type'] === 'application/octet-stream') {
    return { data: {
      data: response.data, title: response.headers['content-disposition']  // 这里是文件名字
    }}
 }

    根据自己下载的文件 的类型去拦截,我这里要下载excel,response的content-type:application/octet-stream,就拦截这个content-type,如果下载的类型较多，可以写个白名单

    const whiteList = ['application/octet-stream']
  if (whiteList.includes(response.headers['content-type'])) {
    return { data: {
      data: response.data, title: response.headers['content-disposition']  // 这里是文件名字
    }}
}

    # 3.在vue文件中下载的方法

          const params = {
          KeywordsType: this.form.keywordsType,
          Keywords: this.form.keyword,
          Status: this.form.status,
          KeFuId: this.form.keFuId,
          PageIndex: this.page.current,
          PageSize: this.page.size
        }
        const res = await http.post(`xxxxxxxxxxx`, params, {}, {
          responseType: 'blob'
        })
        const blob = res.data
        const reader = new FileReader()
        reader.readAsDataURL(blob)
        reader.onload = (e) => {
          const a = document.createElement('a')
          a.download = res.title && res.title.split(';') && (res.title.split(';').length > 1) && decodeURIComponent(res.title.split(';')[1].split('=')[1])
          a.href = e.target.result
          document.body.appendChild(a)
          a.click()
          document.body.removeChild(a)
        }
