过年后从深圳回来已经有一个多月了，发现还是有很多的不同，刚回来就想写下来，记录一下，但是加上天天的加班赶项目和晚上回去又不想写一直拖啊拖，拖到现在。

# 1. 工作时长

第一点比较大的不同是，郑州的公司每周上6天班,工作时间明显比较长,
并且我入职的这家公司居然是996,我也不知道刚回来找工作的时候是怎么想,
除了工资相对还算可以,其他方面和我以前的工作没有任何的优势,
当时可能着急找工作,在加上当时发生的一点事情,让我立即来上班,
我也就过去了,从此就进入了996的泥沼中,想在爬出来可能需要过一段时间了。

# 2.团建

 截止到今天,我上班一个多月了吧,没经历过什么团建吧(好像没有,后续待考证吧),
 面试的时候说每周六下午会有自由活动时间,我上班第一周是周五下午3点左右,
 领导说大家一起下去篮球场打会篮球,算是放松吧,到晚上给我的感觉,
 就像我上小学住校,半个月回家一次,周日下午也是自由活动给看电影就是不准出校门,
 现在是给你打球去附近逛就是不准提前下班回去(要打卡),唯一的好点算是,
 有时候会提供一些零食算是一点福利吧。

# 3.感想

前一段时间github上那位大牛创建了996.icu这个项目, 顿时轰动了国内外所有的程序员,大家像炸了锅一样都纷纷去吐槽自己公司的不好。 中间python之父也询问怎么能够帮到中国的程序员寻求建议, 最后也没有人提出一些有用的建议吧,关闭了讨论区,最后python社区也列了几条, 不接受996公司的资

助不和他们进行合作等,过完清明节回来, 阿里巴巴的大boss,马云针对这个话题把996说成是一种福报, 瞬间批评马云的评论满天飞,再后来京东的刘大佬也入局,在互联网引起了更大的反应, 瞬间吵翻了天,后来比较官网的一些媒体《共青团中央》、中国青年报 也发表了一些评论,事情还在进一步发酵中。 通过这件事我有一天早上睡醒突然想到一个问题,我现在每个月的工作时长和以前差多少, 大概的算了一下,现在的一个月的工作时间(26)是以前(22)的1.5倍还多,看到这个结果, 愣了一会,刚开始我还有点小骄傲的感觉我回来工资很高了。此刻,心里想到姜还是老的辣, 我还是太年轻了。就算这个项目上线后恢复正常,上班时长居然是9小时而不是国家规定的8小时, 我居然是前2天才想到,我的反应真的是够慢的(傻的可以)。

# 4. 总结

任何一件事总会有点收获,我也装作学到一些经验,来点总结:换工作前要了解一下要去的公司的
工作时间和工作的基本信息,体系是不是完整,是不是一个职位让你从前端干到运维所有的事情,
甚至设计都让你来,另一方面就是对每个人都比较关心的薪水,不能只看每个月的薪水,
也要算一下时薪(虽然大多数人都没有牛逼到算时薪的地步),还是大概的比较一下月工资和
每天的薪资甚至没小时的比较一下。遇到加班的公司,不至于傻傻的吃亏太多,自己还傻乐。
