  • 支付宝返回form表单新页面打开

    • vue点击按钮新页面打开一个支付宝的支付页面

    # 由于支付宝返回的是一个form表单

    所以通过axios获取到的是一个包含form表单和script的字符串，我们要在我们的页面自动执行,我们采用的办法

     let dom = document.createElement('div')   // 创建一个div标签
          dom.innerHTML = res.data   //把返回的字符串放到div里面
          dom.id = 'zfbPay'          //给创建的id加一个id
          document.body.before(dom)  //把新建的div插入到body第一个元素
          document.forms[0].style.opacity = 0    //设置为不显示状态,不能让用户看见表单的存在
          document.forms[0].acceptCharset = 'utf-8'  //设置编码,这里我看了支付宝返回的是urf-8,我也使用utf-8
          document.forms[0].target = '_blank' //给form添加target属性，从新页面打开
          document.forms[0].submit()    //提交表单执行form的action
          setTimeout(() => {
            document.getElementById('zfbPay').remove()  // 点击过后删掉创建div
          }, 50)
